Add reCAPTCHA to your website Add reCAPTCHA to your forms and actions you want to protect. method1 //on user interaction //1. Load the JavaScript API with your key. //2. Call grecaptcha.execute for each action you want to protect. Send the response token to your application backend. The response token expires after two minutes. method2 //on an html button //1. Load the JavaScript API with your key. //2. Use a callback function to handle the response token. //3. Add attributes to your HTML button: Send the response token to your application backend. The response token expires after two minutes. Get the response token from the g-recaptcha-response POST parameter when the user submits the form on your site.
